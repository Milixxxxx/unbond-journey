# No-Contact Coach — Crisis / SOS Tools

Recreation of the SOS toolkit referenced in the ebook (High-Load Distraction, Red Cross Stop, Gaslighting Decoder, urge timer).

**Sources:**
- `code_final/Gaslighting Decoder.html` — 20-question assessment
- `code_final/SCHRITT_4_FINAL.html` — SOS step page
- `code_final/bonusE_updated.html` — bonus exercises
- Exercise callouts throughout `UNBOND_Final_v6.html`
