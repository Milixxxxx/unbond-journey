# Landing UI Kit — `unbond.de`

Recreation of the Unbond landing page (hero + problem cards + ToxiCometer CTA + FAQ).

**Source:** `code_final/LANDINGPAGE/index.html` (v17).
**Visual mode:** Dark / Forest — fixed full-bleed hero-bg.png + 160° gradient overlay + glassmorphism cards.
**Type:** Poppins display + Lato body (the prod site uses Poppins for hero while keeping Montserrat fallback). Cream foreground on dark.

## Files
- `index.html` — click-through landing, loads babel + components
- `Hero.jsx` — hero section with badge, H1, CTAs
- `ProblemCards.jsx` — flip-card grid (Gaslighting · Trauma Bond · Narzissmus)
- `ToxiCometerCTA.jsx` — big glass card with the 15-questions promise
- `FAQ.jsx` — accordion
